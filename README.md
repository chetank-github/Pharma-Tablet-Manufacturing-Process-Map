##  Project 1 — Tablet Manufacturing Process Map

### What is this?
A GMP-aligned process flow map showing how a solid dosage
tablet is manufactured from raw material to finished product.

### Process Flow
- START
- Granulation → CPP: Temp · Moisture · Binder Ratio
- Compression → CPP: Hardness · Thickness · Weight
- Coating → CPP: Pan Speed · Temp · Spray Rate
- Packaging → CPP: Seal Integrity · Count · Label
- FINISHED PRODUCT

### Key Concepts Covered
- Critical Process Parameters (CPP)
- In-process QC checkpoints at each stage
- Batch Record documentation points
- GMP manufacturing flow basics

Designed a GMP-aligned process flow map for solid dosage tablet
manufacturing with CPP documentation, QC checkpoints and
batch record traceability.
